# Rock Face Plugin for VS Code

  

Make errors with fun. Powered by Dwayne Johnson.

  

## Features

  

This plugin shows different faces of Dwayne "The Rock" Johnson.

  

Different faces shows on those number of errors:

- First: 0 errors

- Second: 1-2 errors

- Third: 3 - 4 errors

- Fourth: 5+ errors
